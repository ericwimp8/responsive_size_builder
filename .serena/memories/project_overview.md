# Responsive Size Builder - Project Overview

## Purpose
A Flutter package for building responsive UI components that adapt to different screen sizes. The package provides utilities and widgets to handle responsive layouts based on screen breakpoints.

## Tech Stack
- **Language**: Dart
- **Framework**: Flutter (>=1.17.0)
- **SDK**: Dart ^3.5.0
- **Flutter Version**: 3.35.1 (stable channel)
- **Development Tools**:
  - flutter_lints: ^4.0.0
  - very_good_analysis: ^5.1.0
  - custom_lint plugin
  - Dart MCP server for IDE integration
  - Serena for semantic code navigation

## Key Features
- Screen size detection and categorization
- Responsive value selection based on breakpoints
- Multiple builder widgets for different responsive patterns
- Layout constraints handling
- Orientation-aware builders
- Overlay position utilities

## Package Structure
The package exports a comprehensive set of responsive UI building tools through the main `responsive_size_builder.dart` file, providing builders, breakpoint handlers, and utility functions for creating adaptive Flutter applications.
