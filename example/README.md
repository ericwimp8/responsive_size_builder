# responsive_size_builder example

Example Flutter app for `responsive_size_builder`.

The app demonstrates the package's main responsive API families:

- `ScreenWidgetBuilder`
- `ScreenValueBuilder`
- `LayoutWidgetBuilder`
- `LayoutValueBuilder`
- `ScreenSizeWithValue`
- granular breakpoint builders
- orientation-aware builders
- `MaterialResponsiveSize`
- `ResponsiveGrid`
- `ResponsiveSectionRow`

Run the example from this directory:

```sh
flutter run
```

Resize the app window or run it on different devices to see each page resolve
different breakpoints, values, and layout structures.
